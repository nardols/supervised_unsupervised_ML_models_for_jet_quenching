# Study of jet modification in relativistic heavy-ion collisions with machine learning techniques

This repository contains materials and code developed during my Master’s research, including simulations, analysis pipelines, machine learning architectures, and the data used in all the analysis.

This project was strongly inspired by the following works: “Identifying quenched jets in heavy ion collisions with machine learning” by Lihan Liu, Julia Velkovska, Yilun Wu, and Marta Verweij, and “Jet substructure observables for jet quenching in Quark–Gluon Plasma: a Machine Learning driven analysis” by Miguel Crispim Romão, José Guilherme Milhano, and Marco van Leeuwen. They provided key references for the scientific motivation and methodology, and had a huge influence on developing the codes.

The data used in this work is available at https://drive.google.com/drive/folders/1PRXPdcYaZ55PJOt2gs75muheoctf5DSK?usp=share_link
